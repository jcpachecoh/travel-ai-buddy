## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## How to Use

1. Enter your desired destination
2. Specify the duration of your trip
3. Select your budget range
4. Share your interests and preferences
5. Click "Generate Travel Plan" to get your AI-powered itinerary
